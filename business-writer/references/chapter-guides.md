# Chapter Guides - 章节专属写作指南

## 概述

本文档提供商用计划书各章节的专属写作指南，包括：
- 执行摘要
- 行业分析
- 财务分析
- 竞争分析
- 战略分析
- 风险分析

---

## 一、执行摘要

### 定位

投资人的"第一眼"，30 秒内建立价值认知。

### 结构模板

**段落 1（公司定位，2-3 句话）**
```
[公司名] 是 [细分领域] 的 [市场地位描述]，
核心业务为 [主营业务一句话]，目前处于 [发展阶段]。
```

**段落 2（财务亮点，3-4 句话）**
```
[最近财年] 实现营收 [X] 亿元（+YoY%），净利润 [Y] 亿元（+YoY%）。
核心指标方面，[指标 1] 为 [数值]（[对比]），
[指标 2] 为 [数值]（[对比]），反映出 [一个核心判断]。
```

**段落 3（投资亮点，3-5 点，每点 1-2 句话）**
```
核心投资亮点包括：
（1）[优势 1]：[具体表现 + 数据支撑]；
（2）[优势 2]：[具体表现 + 数据支撑]；
（3）[优势 3]：[具体表现 + 数据支撑]。
```

**段落 4（风险与建议，2-3 句话）**
```
主要风险在于 [最大风险点]。综合以上分析，
建议 [投资评级]，目标估值区间 [X-Y] 亿元。
```

### 示例

```
海尔智家是智能家电领域的领军企业，核心业务为智能家电产品的研发、
生产和销售，目前处于全球化扩张阶段。

2024 年公司实现营收 138 亿元（+17.1%），净利润 11.7 亿元（+19.3%）。
核心指标方面，ROE 为 15.2%（同比 +1.1pct），为近五年最高水平；
毛利率 33.2%（同比 +1.4pct），反映出盈利能力持续改善。

核心投资亮点包括：
（1）海外市场高速增长：出口收入同比增长 32%，占总营收比例提升至 25%；
（2）智能产品线爆发：智能家电营收同比增长 35%，成为核心增长引擎；
（3）盈利质量优异：毛利率为近五年最高，增长并非以牺牲利润为代价。

主要风险在于海外贸易政策的不确定性。综合以上分析，
建议"买入"，目标估值区间 180-200 亿元。
```

### 质量检查

- [ ] 字数：300-500 字
- [ ] 语气：自信、果断、有判断力
- [ ] 数据：只列最核心的 3-5 个指标
- [ ] 结论：明确的投资建议

## 二、行业分析

### 定位

回答"这是不是一个好赛道"。

### 段落类型 A：市场规模

**模板**：
```
[行业] 市场规模在 [年份] 达到 [X] 亿元，
[增长/下降] [Y]%（来源：[权威机构]）。
从增长驱动力来看，[驱动因素 1] 贡献了约 [Z]% 的增量，
具体表现为 [具体数据/案例]。
[驱动因素 2] 同样起到重要推动作用，[具体说明]。
展望 [未来年份]，市场规模有望达到 [预测值]，
CAGR 约 [Z]%，增长动力主要来自 [1-2 个核心因素]。
```

**示例**：
```
中国智能家电市场规模在 2024 年达到 1,850 亿元，同比增长 22.3%
（来源：中国家电协会《2024 年智能家电行业报告》）。从增长驱动力
来看，消费升级贡献了约 60% 的增量，具体表现为一二线城市
智能家电渗透率从 2020 年的 15% 提升至 2024 年的 38%。

房地产精装配套同样起到重要推动作用，2024 年精装房智能家电
配套率从 2020 年的 8% 提升至 35%，贡献了约 30% 的增量。

展望 2027 年，市场规模有望达到 3,500 亿元，CAGR 约 24%，
增长动力主要来自：（1）下沉市场渗透率提升（当前不足 10%）；
（2）全屋智能场景驱动（从单品向场景升级）。
```

### 段落类型 B：竞争格局

**模板**：
```
当前 [行业] 呈现 [集中/分散] 格局，CR5 为 [X]%，
较 [上年/两年前] 的 [Y]% [提升/下降] [Z] 个百分点。
第一梯队为 [公司 A] 和 [公司 B]，合计市占率 [X]%；
第二梯队包括 [公司 C、D、E]，市占率在 [区间]。
[一个趋势判断]，预计 [时间] CR5 将提升至 [目标值]，
主要得益于 [原因]。
```

**示例**：
```
当前智能家电行业呈现"一超多强"格局，CR5 为 58%，较 2020 年的
45% 提升 13 个百分点，集中度加速提升。第一梯队为美的集团和
海尔智家，合计市占率 35%；第二梯队包括格力、小米、海信，
市占率在 5%-10% 区间。

行业集中度提升趋势明确，预计 2027 年 CR5 将提升至 70%，
主要得益于：（1）头部企业研发投入优势（头部研发费用率是
中小企业的 3-5 倍）；（2）渠道变革加速（线上+新零售对
传统渠道的冲击）；（3）智能化门槛提升（AIoT 技术投入
门槛高）。
```

### 质量检查

- [ ] 数据来源权威（行业协会、权威咨询机构）
- [ ] 驱动力量化（贡献度百分比）
- [ ] 预测有依据（基于驱动力推导）
- [ ] 竞争格局清晰（梯队划分明确）

## 三、财务分析

### 定位

用数据验证公司质量，是全文最核心的章节。

### 固定结构

```
子章节标题（主旨式）
  │
  ├── 第 1 段：当前水平 + 对比（What）
  │     → 当前数值 + 同比变化 + 行业对比
  │
  ├── 第 2 段：原因分析（Why）
  │     → 变化原因拆解（分主次）
  │     → 每个原因尽量量化
  │
  ├── 第 3 段：深层含义（How）
  │     → 这个变化意味着什么
  │     → 对公司竞争力/投资价值的影响
  │
  ├── 第 4 段：趋势展望（可选）
  │     → 未来走势预判
  │     → 关键假设和风险点
  │
  └── 图表引用 + 解读（300 字+）
```

### 五大维度写作要点

#### 1. 盈利能力

**必须回答**：
- ROE 提升/下降靠什么驱动？
- 质量如何？可持续吗？
- 与同业对比如何？

**禁止**：只列 ROE 数字不做杜邦分解

**示例结构**：
```
2024 年公司 ROE 为 15.2%，较上年提升 1.1 个百分点，为近五年
最高水平（来源：公司年报）。杜邦分解显示，毛利率改善（+1.4pct）
是主要驱动因素，贡献了约 0.7 个百分点的 ROE 提升；资产周转率
从 1.2 次提升至 1.3 次，贡献了约 0.3 个百分点；权益乘数基本
持平（1.85 vs 1.83）。

与同业对比，公司 ROE 低于美的集团（22.5%），但高于格力电器
（13.8%）和行业平均（12.5%）。差距主要来自资产周转率
（美的为 1.6 次），反映出公司在资产运营效率方面仍有提升空间。

展望 2025 年，ROE 有望提升至 16%-17% 区间，主要得益于：
（1）智能产品占比提升继续拉动毛利率；（2）产能利用率爬坡
提升资产周转率。主要风险在于原材料价格反弹和汇率波动。
```

#### 2. 营运能力

**必须回答**：
- 周转率变化反映了什么经营变化？
- 效率提升/恶化的原因？
- 与业务规模匹配吗？

**禁止**：只说"存货周转率提升"不说原因

#### 3. 偿债能力

**必须回答**：
- 债务结构是否健康？
- 短期有流动性风险吗？
- 融资渠道畅通吗？

**禁止**：只列资产负债率不做结构分析

#### 4. 成长能力

**必须回答**：
- 增长是内生的还是外延的？
- 增长质量如何？
- 增长可持续吗？

**禁止**：只说"营收增长 20%"不做归因

#### 5. 现金流

**必须回答**：
- 利润的"含金量"如何？
- FCF 是正还是负？为什么？
- 现金转化周期变化趋势？

**禁止**：只列现金流数字不做质量判断

### 质量检查

- [ ] 每个维度都有杜邦分解或因素拆解
- [ ] 有同业对比（至少 3 家可比公司）
- [ ] 趋势展望有明确假设
- [ ] 图表解读 300 字以上

## 四、竞争分析

### 定位

回答"公司凭什么能赢"。

### 竞争对手描述模板

```
[竞品公司名] 是 [市场地位]，2024 年实现营收 [X] 亿元，
毛利率 [Y]%，市占率 [Z]%。其核心优势在于 [优势 1-2 点]，
主要体现在 [具体数据/事实]。但同时存在 [劣势 1-2 点]，
具体表现为 [具体数据/事实]。与本公司相比，
[直接对比：我方强在哪/弱在哪]。
```

### SWOT 写法

**禁止**：做成表格就完事

**正确写法**：

**优势（Strengths）**
```
优势方面，公司在 [领域] 具备显著的竞争壁垒。具体来看，
[优势 1] 体现在 [量化数据]，远高于行业平均的 [对比数据]；
[优势 2] 则通过 [事实] 得以验证。
```

**劣势（Weaknesses）**
```
劣势方面，[劣势 1] 是当前最突出的短板，[具体影响量化]。
建议通过 [具体措施] 予以补强。
```

**机会（Opportunities）**
```
机会方面，[市场机会 1] 为公司提供了 [量化空间] 的增长潜力，
[具体说明如何抓住]。[市场机会 2] 同样值得关注，[说明]。
```

**威胁（Threats）**
```
威胁方面，[最大威胁] 是最需要警惕的风险点，[量化影响]。
公司已通过 [应对措施] 予以缓释，但仍需持续关注。
```

### 质量检查

- [ ] 每个竞争对手都有数据支撑
- [ ] SWOT 展开论述，不是表格
- [ ] 劣势有改进措施
- [ ] 与竞品对比有明确结论

## 五、战略分析

### 定位

回答"公司未来怎么走，能不能走成"。

### 战略目标写法（SMART 原则）

**❌ 空洞示例**：
```
公司将继续深耕主业，拓展海外市场，提升品牌影响力。
```

**✅ 具体示例**：
```
短期（2025 年）：营收目标 160 亿元（+16%），其中海外收入占比
从 25% 提升至 30%；毛利率维持在 33% 以上。

中期（2026-2027 年）：完成东南亚第二工厂建设，海外本地化
产能占比达到 40%；智能产品线营收占比从 15% 提升至 30%。

长期（2028 年+）：进入行业全球前五，海外营收占比超过 40%。
```

### 战略路径

**必须包含**：
1. 具体举措（做什么）
2. 资源投入（投多少）
3. 时间节点（何时完成）
4. 预期效果（达成什么）

**示例**：
```
为实现上述目标，公司将重点推进三大战略举措：

（1）海外产能布局：2025-2026 年投资 15 亿元建设东南亚第二工厂，
设计产能 100 万台/年，预计 2027 年投产。投产后海外本地化产能
占比将从 20% 提升至 40%，物流成本下降约 15%，交货周期缩短 40%。

（2）智能产品线扩展：2025 年研发投入 8 亿元（占营收 5%），
扩展至智能厨房和智能卫浴两个新场景，预计新增营收 5-7 亿元。
同时与华为、小米等生态平台深度合作，接入其智能家居生态系统。

（3）渠道下沉：2025 年在三四线城市新增 500 家专卖店，
覆盖率从 45% 提升至 60%。单店投资约 30 万元，总投资 1.5 亿元，
预计贡献增量营收 8-10 亿元。
```

### 质量检查

- [ ] 目标符合 SMART 原则
- [ ] 路径清晰、可执行
- [ ] 资源投入具体
- [ ] 风险点有应对预案

## 六、风险分析

### 定位

诚实披露风险，给出应对预案。

### 风险描述结构

```
[风险名称]：[发生概率评估] [影响程度量化]
[具体说明风险来源和传导机制]
[已采取的缓释措施]
[剩余风险和应对预案]
```

### 示例

```
（1）海外贸易政策风险：【发生概率：中】【影响程度：高】

公司在东南亚地区的营收占比已达 25%，若主要出口国（美国、欧盟）
加征关税或设置贸易壁垒，将直接影响出口业务。据测算，若美国
对东南亚产家电产品加征 25% 关税，将影响公司营收约 15 亿元，
影响净利润约 2 亿元。

公司已通过以下措施缓释：①加速推进海外本地化生产，降低
"中国制造"标签风险；②拓展"一带一路"沿线国家市场，降低
对欧美市场依赖（目标从 70% 降至 60%）；③购买出口信用保险，
覆盖 80% 的关税风险。

剩余风险：若全球贸易保护主义全面升级，仍可能对海外业务
造成重大冲击。应对预案：①加快海外并购，通过收购当地品牌
规避贸易壁垒；②建立全球供应链预警机制，提前 6 个月监测
贸易政策变化。
```

### 质量检查

- [ ] 风险按重要性排序
- [ ] 发生概率和影响程度量化
- [ ] 缓释措施具体
- [ ] 剩余风险有预案
