# 质量检查清单（v4.2 强制版）

## 关口 3：整合完成检查（12项强制检查）

### 🚨 绝对禁止项（违反即失败）
- [ ] **专家内容完整性**：所有专家内容100%保留，无任何删减
- [ ] **无内容重写**：未添加/修改/删除任何专家内容  
- [ ] **无内容添加**：未在专家内容基础上添加任何分析、总结或评论
- [ ] **图表位置正确**：所有图表已在各章节中直接嵌入，无单独成章

### ✅ 强制执行项
- [ ] **直接拼接操作**：使用文件合并命令直接拼接，未进行内容审查
- [ ] **标题编号统一**：使用1.1, 2.1, 3.1, 4.1, 5.1, 6.1格式
- [ ] **数据来源移除**：文中所有数据来源标注已移除
- [ ] **参考文献添加**：文末有统一参考文献章节，按类型分组
- [ ] **图片路径正确**：所有图片路径指向charts/目录
- [ ] **专业技能调用**：已调用Word/PDF文档生成技能
- [ ] **无硬编码输出**：未使用python-docx等冒充专业技能
- [ ] **质量评分达标**：整合后质量评分 >= 80分

## 关口 4：最终检查（15项强制检查）

### 📊 基础指标检查
- [ ] **总字数 ≥20000字**：专家原始内容总和达标
- [ ] **总图表 ≥10张**：全部直接嵌入相关文本
- [ ] **总表格达标**：每章包含2-3个专业表格
- [ ] **章节完整**：执行摘要 + 7大章节 + 参考文献

### 📝 内容质量检查  
- [ ] **论文式写作**：无提纲式1.2.3列表，全部段落论述
- [ ] **三层论述结构**：What-Why-How分析深度达标
- [ ] **主旨式标题**：无数据堆砌到标题
- [ ] **语言质量**：无AI味（Humanizer已调用）

### 🔍 数据与格式检查
- [ ] **数据一致性**：所有财务数据完全一致
- [ ] **参考文献完整**：文末参考文献格式规范、分类清晰
- [ ] **第七章完整**：包含投资评级、核心逻辑、风险提示、目标价位
- [ ] **格式规范**：自动生成目录、统一排版
- [ ] **无提示词泄露**：无"本段3200字"等内部指令

1. 质量审核官一票否决
2. 项目立即打回重做  
3. 流程监管专家启动惩罚机制
