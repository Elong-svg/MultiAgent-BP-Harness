# Business Writer - 商业写作引擎 v1.0

## 技能定位

本技能不是代码工具，而是一套**给 AI 的写作方法论 + 模板 + 示例**。AI 读完后，应具备将原始数据和分析结果转化为专业商业论述的能力。

**核心目标**：让 AI 写出像人写的、有深度的商业分析内容，而不是冷冰冰的数据堆砌。

---

## 一、核心理念

### 1.1 这个技能解决什么问题

| 现状问题 | 解决方案 |
|---------|---------|
| AI 写出来的东西一眼假，全是"综上所述""由此可见" | 提供**过渡句库**和**连接词矩阵**，消灭 AI 味 |
| 每段话都是"数据 + 数据 + 数据"，没有分析 | 强制**三层结构**：事实→原因→含义 |
| 标题要么太长（堆数据），要么太短（没信息） | **主旨式标题公式**：动作 + 对象 + 角度 |
| 段落之间没有逻辑，像拼凑的 | **段落衔接规则**：上一段的结论=下一段的起点 |
| 全文语气不统一，有的像教科书有的像新闻稿 | **语气锚定**：每章开头给一个"语气样本" |

### 1.2 写作的三个底层原则

**原则一：写给投资人看，不是写给教授看**
- → 不要学术腔，要"有数据支撑的商业判断"
- → 每段话最后应该有一个"so what"（所以呢？这说明什么？）

**原则二：数据是论据，不是结论**
- → 永远先给结论，再用数据支撑
- → 禁止："2023 年营收 100 亿，2024 年营收 120 亿，同比增长 20%"
- → 应该："营收增速加快，2024 年同比增长 20% 至 120 亿元，较上年提升 5 个百分点，主要受益于海外市场放量和新品贡献"

**原则三：每一段都要能独立回答一个问题**
- → 读者只看这一段，也能理解一个完整的分析点
- → 不依赖上下文就能看懂

## 二、输入与输出规范

### 2.1 输入格式

当调用 business-writer 时，它会收到以下结构化输入：

```yaml
input:
  chapter: "财务分析"              # 目标章节
  section: "盈利能力分析"          # 子章节
  data:                           # 原始数据（结构化）
    revenue:
      - year: 2020, value: 80, unit: "亿元"
      - year: 2021, value: 92, unit: "亿元"
      - year: 2022, value: 105, unit: "亿元"
      - year: 2023, value: 120, unit: "亿元"
      - year: 2024, value: 138, unit: "亿元"
    gross_margin:
      - year: 2020, value: 28.5
      - year: 2021, value: 29.2
      - year: 2022, value: 30.1
      - year: 2023, value: 31.8
      - year: 2024, value: 33.2
    peer_comparison:               # 同业对比数据
      company: 33.2
      industry_avg: 29.5
      leader: 38.0
  context:                        # 写作上下文
    company: "海尔智家"
    industry: "家用电器"
    audience: "investor"          # 目标读者
    style: "investment_report"    # 风格定位
  requirements:                   # 特殊要求
    min_words: 800
    must_include: ["同比分析", "行业对比", "趋势判断"]
    charts_available: ["revenue_trend.png", "margin_comparison.png"]
```

### 2.2 输出格式

```yaml
output:
  section_title: "3.1 营收增长趋势与驱动因素分析"  # 主旨式标题
  paragraphs:                     # 正文段落数组
    - paragraph_1
    - paragraph_2
    - ...
  chart_references:               # 图表引用
    - chart: "revenue_trend.png"
      position: "paragraph_2 之后"
      caption: "图 3-1：近五年营业收入趋势"
      interpretation: "300 字以上的图表解读"
  word_count: 1200
  quality_flags:                  # 质量标记
    has_what_why_how: true
    has_comparison: true
    has_trend_analysis: true
    no_ai_patterns: true
```

## 三、核心写作框架

### 3.1 三层论述结构（最核心的规则）

**每个分析段落必须包含三层：**

```
┌─────────────────────────────────────────────┐
│  第一层：WHAT（事实层）                        │
│  → 陈述数据事实，给出具体数值                  │
│  → 与基准对比（同比/环比/行业/竞品）            │
│  → 字数占比：约 25%                            │
├─────────────────────────────────────────────┤
│  第二层：WHY（归因层）                         │
│  → 分析变化背后的原因                          │
│  → 分主次，先说主要原因                        │
│  → 尽量量化每个原因的贡献度                     │
│  → 字数占比：约 40%                            │
├─────────────────────────────────────────────┤
│  第三层：HOW（含义层）                         │
│  → 这个数据意味着什么                          │
│  → 对公司战略/投资价值的影响                    │
│  → 未来趋势预判                                │
│  → 字数占比：约 35%                            │
└─────────────────────────────────────────────┘
```

**实际写作示例（对比）：**

❌ **不合格写法（只有 What）：**
> 2024 年公司实现营收 138 亿元，同比增长 17.1%。毛利率 33.2%，较上年提升 1.4 个百分点。

✅ **合格写法（What-Why-How 三层）：**
> 2024 年公司实现营收 138 亿元，同比增长 17.1%，增速较上年的 14.3% 提升 2.8 个百分点，**在可比同业中位列第一梯队**（来源：Wind，家电行业数据库）。营收增长的加速并非偶然——**从驱动因素拆解来看**，海外市场贡献了约 40% 的增量，受益于东南亚工厂 Q2 投产后的本地化成本优势；国内市场方面，智能家电新品线实现营收 22 亿元，同比增长 35%，成为内销增长的核心引擎。**值得注意的是**，营收增速的提升伴随着毛利率的同步改善（33.2%，同比 +1.4pct），这意味着增长并非以牺牲盈利能力为代价的"虚胖"，而是有质量的扩张。**展望 2025 年**，随着海外产能利用率爬坡和智能产品线的持续放量，营收增速有望维持在 15%-20% 区间。

### 3.2 标题写作公式

**禁止**：数据堆砌到标题（如"3.1 营收五年复合增长 18.5%"）
**禁止**：空洞标题（如"3.1 财务分析"）

**公式：序号 + 分析动作 + 分析对象 + 分析角度**

| 章节 | 合格标题示例 |
|------|------------|
| 行业分析 | 2.1 家电行业规模测算与增长驱动因素拆解 |
| 行业分析 | 2.2 产业链价值分布与利润池迁移趋势 |
| 竞争分析 | 3.1 竞争格局演变：从分散到集中的结构性拐点 |
| 财务分析 | 4.1 盈利能力改善：毛利率提升的可持续性分析 |
| 财务分析 | 4.2 ROE 提升路径：杜邦分解视角下的质量评估 |
| 战略分析 | 5.1 国际化战略：从"出海"到"全球化"的关键跨越 |
| 风险分析 | 6.1 客户集中度风险：前五大客户依赖度量化评估 |

### 3.3 段落衔接规则

**核心原则：上一段的结尾，是下一段的开头。**

```markdown
段落 1 结尾：...毛利率提升主要受益于产品结构优化和原材料成本下降。

段落 2 开头：在毛利率改善的同时，费用端的变化同样值得关注——
           2024 年销售费用率从 12.5% 上升至 14.0%...

段落 2 结尾：...研发投入的加码虽然短期压缩利润，但为长期技术壁垒奠定基础。

段落 3 开头：将营收增长和费用变化综合来看，净利率的表现更能反映
           公司的真实盈利质量——
```

**衔接词矩阵（按功能分类）：**

| 功能 | 可用的衔接表达 |
|------|--------------|
| **递进** | 在...的同时、进一步看、更值得关注的是、无独有偶 |
| **转折** | 然而、但值得注意的是、与此形成对比的是、另一方面 |
| **因果** | 因此、由此、基于以上分析、这背后反映的是 |
| **总结** | 综合来看、总体而言、从以上分析可以得出 |
| **举例** | 具体来看、以...为例、从细分数据来看 |
| **对比** | 与之相比、同业对比显示、行业平均而言 |
| **递进到未来** | 展望未来、基于以上趋势、如果这一态势延续 |

**禁用的 AI 味过渡词**（出现即扣分）：
- ❌ "综上所述"
- ❌ "由此可见"
- ❌ "总而言之"
- ❌ "不难看出"
- ❌ "值得一提的是"（过度使用）
- ❌ "在当今...的大背景下"

### 3.4 数据引用规范

**原则：数据永远服务于观点，不是观点服务于数据。**

```markdown
❌ 错误：2024 年营收 138 亿元（YoY+17.1%），毛利率 33.2%（YoY+1.4pct），
        净利率 8.5%（YoY+0.3pct），ROE15.2%（YoY+1.1pct）。

✅ 正确：2024 年核心盈利指标全面改善，其中 ROE 提升至 15.2%（+1.1pct），
        为近五年最高水平。拆解来看，毛利率改善（33.2%，+1.4pct）是最
        主要的贡献因素，而净利率的提升幅度（+0.3pct）相对温和，反映出
        费用端的增长部分抵消了毛利改善的效果。
```

**数据来源标注格式：**
- 年报数据：`（来源：公司 2024 年年报，第 X 页）`
- 第三方数据：`（来源：艾瑞咨询《2024 年 XX 行业报告》）`
- 自主计算：`（来源：根据公司年报数据计算）`
- 同行对比：`（来源：各公司 2024 年年报，Wind 整理）`

## 四、章节专属写作指南

### 4.1 执行摘要

**定位**：投资人的"第一眼"，30 秒内建立价值认知。

**结构模板：**

```
段落 1（公司定位，2-3 句话）：
  [公司名] 是 [细分领域] 的 [市场地位描述]，
  核心业务为 [主营业务一句话]，目前处于 [发展阶段]。

段落 2（财务亮点，3-4 句话）：
  [最近财年] 实现营收 [X] 亿元（+YoY%），净利润 [Y] 亿元（+YoY%）。
  核心指标方面，[指标 1] 为 [数值]（[对比]），
  [指标 2] 为 [数值]（[对比]），反映出 [一个核心判断]。

段落 3（投资亮点，3-5 点，每点 1-2 句话）：
  核心投资亮点包括：（1）[优势 1]：[具体表现 + 数据支撑]；
  （2）[优势 2]：[具体表现 + 数据支撑]；（3）...

段落 4（风险与建议，2-3 句话）：
  主要风险在于 [最大风险点]。综合以上分析，
  建议 [投资评级]，目标估值区间 [X-Y] 亿元。
```

**字数**：300-500 字  
**语气**：果断、自信、有判断力

### 4.2 行业分析

**定位**：回答"这是不是一个好赛道"。

**段落类型及写法：**

**类型 A：市场规模段**
```
写法模板：
[行业] 市场规模在 [年份] 达到 [X] 亿元，
[增长/下降] [Y]%（来源：[权威机构]）。
从增长驱动力来看，[驱动因素 1] 贡献了约 [Z]% 的增量，
具体表现为 [具体数据/案例]。
[驱动因素 2] 同样起到重要推动作用，[具体说明]。
展望 [未来年份]，市场规模有望达到 [预测值]，
CAGR 约 [Z]%，增长动力主要来自 [1-2 个核心因素]。
```

**类型 B：竞争格局段**
```
写法模板：
当前 [行业] 呈现 [集中/分散] 格局，CR5 为 [X]%，
较 [上年/两年前] 的 [Y]% [提升/下降] [Z] 个百分点。
第一梯队为 [公司 A] 和 [公司 B]，合计市占率 [X]%；
第二梯队包括 [公司 C、D、E]，市占率在 [区间]。
[一个趋势判断]，预计 [时间] CR5 将提升至 [目标值]，
主要得益于 [原因]。
```

### 4.3 财务分析（重点章节）

**定位**：用数据验证公司质量，是全文最核心的章节。

**每个子章节的固定结构：**

```
子章节标题（主旨式）
  │
  ├── 第 1 段：当前水平 + 对比（What）
  │     → 当前数值 + 同比变化 + 行业对比
  │
  ├── 第 2 段：原因分析（Why）
  │     → 变化原因拆解（分主次）
  │     → 每个原因尽量量化
  │
  ├── 第 3 段：深层含义（How）
  │     → 这个变化意味着什么
  │     → 对公司竞争力/投资价值的影响
  │
  ├── 第 4 段：趋势展望（可选）
  │     → 未来走势预判
  │     → 关键假设和风险点
  │
  └── 图表引用 + 解读（300 字+）
```

**五大维度的写作要点：**

| 维度 | 必须回答的问题 | 禁止的写法 |
|------|--------------|-----------|
| **盈利能力** | ROE 提升/下降靠什么驱动？质量如何？可持续吗？ | 只列 ROE 数字不做杜邦分解 |
| **营运能力** | 周转率变化反映了什么经营变化？效率提升/恶化的原因？ | 只说"存货周转率提升"不说原因 |
| **偿债能力** | 债务结构是否健康？短期有流动性风险吗？ | 只列资产负债率不做结构分析 |
| **成长能力** | 增长是内生的还是外延的？增长质量如何？ | 只说"营收增长 20%"不做归因 |
| **现金流** | 利润的"含金量"如何？FCF 是正还是负？为什么？ | 只列现金流数字不做质量判断 |

### 4.4 竞争分析

**定位**：回答"公司凭什么能赢"。

**竞争对手描述模板：**
```
[竞品公司名] 是 [市场地位]，2024 年实现营收 [X] 亿元，
毛利率 [Y]%，市占率 [Z]%。其核心优势在于 [优势 1-2 点]，
主要体现在 [具体数据/事实]。但同时存在 [劣势 1-2 点]，
具体表现为 [具体数据/事实]。与本公司相比，
[直接对比：我方强在哪/弱在哪]。
```

**SWOT 段落写法（禁止做成表格就完事）：**
```
❌ 错误：做一个 SWOT 四格表格，每个格子写两三句话。

✅ 正确：
优势方面，公司在 [领域] 具备显著的竞争壁垒。具体来看，
[优势 1] 体现在 [量化数据]，远高于行业平均的 [对比数据]；
[优势 2] 则通过 [事实] 得以验证。

劣势方面，[劣势 1] 是当前最突出的短板，[具体影响量化]。
建议通过 [具体措施] 予以补强。
```

### 4.5 战略分析

**定位**：回答"公司未来怎么走，能不能走成"。

**战略目标写法（SMART 原则）：**
```
❌ 空洞："公司将继续深耕主业，拓展海外市场，提升品牌影响力。"

✅ 具体：
短期（2025 年）：营收目标 160 亿元（+16%），其中海外收入占比
从 25% 提升至 30%；毛利率维持在 33% 以上。

中期（2026-2027 年）：完成东南亚第二工厂建设，海外本地化
产能占比达到 40%；智能产品线营收占比从 15% 提升至 30%。

长期（2028 年+）：进入行业全球前五，海外营收占比超过 40%。
```

## 五、语言风格控制

### 5.1 语气锚定

每种章节有不同的语气要求：

| 章节 | 语气 | 示例关键词 |
|------|------|-----------|
| 执行摘要 | 自信、果断、有判断力 | "核心亮点""关键驱动""建议关注" |
| 行业分析 | 客观、宏观、有格局 | "从结构来看""长期趋势""驱动因素" |
| 财务分析 | 精确、冷静、有洞察 | "拆解来看""值得注意的是""深层反映" |
| 竞争分析 | 锐利、直接、有立场 | "相比之下""核心差距""护城河在于" |
| 战略分析 | 前瞻、务实、有逻辑 | "基于以上分析""关键假设""实施路径" |
| 风险分析 | 诚实、谨慎、有预案 | "需警惕的是""概率评估""缓释措施" |

### 5.2 消灭 AI 味的具体技巧

```
技巧 1：避免"总 - 分 - 总"的八股结构
  → 不要每段都"首先...其次...最后..."
  → 用自然的逻辑推进替代列举

技巧 2：避免过度使用"的"
  → "公司的盈利能力的持续提升的背后的驱动因素" ← 烂
  → "公司盈利能力持续提升，背后驱动因素在于" ← 好

技巧 3：避免万能句式
  → "随着...的不断发展" ← AI 味浓
  → 直接说事实，不用铺垫

技巧 4：加入"不完美"的表达
  → "但这一策略也存在隐忧"
  → "需要指出的是"
  → "不过，这一判断需要两个前提"

技巧 5：段落长短要有变化
  → 不要每段都是 4-5 行
  → 重点段落可以长（8-10 行）
  → 过渡段可以短（2-3 行）
  → 重要结论可以单独成段（1-2 行）
```

❌ "随着 XX 的不断发展"
❌ "XX 具有广阔的发展前景"
❌ "XX 公司是一家致力于 XX 的领先企业"

❌ "公司管理层对未来发展充满信心"
❌ "XX 行业迎来了前所未有的发展机遇"

## 六、质量自检清单

写完每个章节后，用以下清单自检。**任何一项不合格都必须重写该段落。**

### 6.1 段落级检查（每段都要过）

```
□ 三层结构：这段话是否包含 What-Why-How 三层？
□ 数据支撑：每个观点是否有具体数据支撑？
□ 来源标注：所有数据是否标注了来源？
□ 对比维度：是否包含至少一种对比（同比/环比/行业/竞品）？
□ 独立可读：只看这一段，能否理解一个完整的分析点？
□ AI 味检查：是否避免了禁止句式？
□ 衔接自然：上一段和这一段之间是否有逻辑衔接？
```

### 6.2 章节级检查

```
□ 字数达标：是否达到最低字数要求？
□ 标题主旨：标题是否是"动作 + 对象 + 角度"格式？
□ 图表嵌入：是否嵌入了可用的图表并配了解读？
□ 逻辑完整：从开头到结尾是否有清晰的逻辑线？
□ 语气一致：全章语气是否统一？
□ 无重复内容：是否有和前面章节重复的论述？
```

### 6.3 全文级检查

```
□ 数据一致性：同一数据在全文各处是否一致？
□ 编号连续：章节编号是否连续无跳号？
□ 引用对应：图表编号和正文引用是否对应？
□ 结论一致：执行摘要的结论和各章分析是否一致？
```

## 七、示例库

### 7.1 营收分析完整示例

```markdown
#### 4.1 营收增长趋势与驱动因素分析

2024 年公司实现营业收入 138.2 亿元，同比增长 17.1%，增速较上年
提升 2.8 个百分点，**在 A 股家电板块可比公司中排名前三**（来源：
Wind，家电行业数据库）。回顾近五年走势，营收从 2020 年的 80.3
亿元增长至 2024 年的 138.2 亿元，CAGR 达 14.5%，呈现稳健增长态势
（参见图 4-1）。

从增长结构来看，2024 年营收增量（约 20.2 亿元）主要来源于三个
方面。**海外市场是最大的增量贡献者**，出口收入同比增长 32% 至
34.5 亿元，占总营收比例从 2023 年的 21% 提升至 25%。这一增长主要
受益于东南亚工厂于 Q2 投产后的本地化生产优势——物流成本下降约
15%，交货周期缩短 40%，使得公司在价格和响应速度上同时获得
竞争力。**国内市场方面**，智能家电新品线表现亮眼，全年实现
营收 22.3 亿元，同比增长 35%，成为内销增长的核心引擎。具体来看，
智能门锁、智能窗帘等新品类贡献了约 8 亿元的增量，验证了公司
"从单品到全屋智能"战略的有效性。**传统品类则保持稳健**，
冰箱、洗衣机等核心品类合计营收 78 亿元，同比增长 8%，跑赢行业
平均增速（5%，来源：中国家电协会），反映出公司在国内存量
市场的份额仍在稳步提升。

值得注意的是，营收增速的加快并未以牺牲盈利质量为代价。2024 年
毛利率为 33.2%，较上年提升 1.4 个百分点，这意味着增长是"有利润
的增长"而非靠降价换量。从现金转化角度看，经营现金流净额为
15.8 亿元，与营收的比值为 11.4%，现金回款质量良好。

展望 2025 年，营收有望维持 15%-20% 的增长区间。核心驱动力包括：
（1）东南亚工厂产能利用率预计从当前的 60% 提升至 85%，贡献增量
收入约 8 亿元；（2）智能产品线将扩展至智能厨房和智能卫浴两个
新场景，预计新增营收 5-7 亿元；（3）国内市场通过渠道下沉，
三四线城市覆盖率从 45% 提升至 60%。主要风险点在于海外贸易政策
的不确定性，以及国内房地产市场低迷对工程渠道的拖累。

【图 4-1：近五年营业收入趋势】
（图表解读：2020-2024 年营收从 80.3 亿元增长至 138.2 亿元，
年均复合增长率 14.5%。2021 年增速最快（+14.6%），2022 年受
宏观环境影响增速放缓（+14.1%），2023-2024 年重回加速通道
（+14.3%/+17.1%），增长动能持续增强。）
```

### 7.2 毛利率分析完整示例

```markdown
#### 4.2 盈利能力核心：毛利率改善的可持续性评估

2024 年公司综合毛利率为 33.2%，较上年的 31.8% 提升 1.4 个百分点，
为近五年最高水平，但**仍低于行业龙头美的集团 38.0% 的水平**
（来源：美的集团 2024 年年报），差距约 4.8 个百分点。

毛利率的改善可归因于三个因素，按贡献度排序如下。**第一，产品
结构优化是最大贡献因素**，高毛利的智能产品线占比从 12% 提升至
16%，拉动综合毛利率约 0.8 个百分点。智能产品的毛利率约为 42%，
较传统产品（约 30%）高出 12 个百分点，结构升级的边际效应显著。
**第二，原材料成本下降贡献了约 0.4 个百分点**，核心原材料中
钢材价格同比下降 8%（从 4,200 元/吨降至 3,860 元/吨），塑料粒子
价格下降 5%，直接降低了单位制造成本。**第三，产能利用率提升
贡献了约 0.2 个百分点**，随着 Q2 东南亚工厂投产和国内产线优化，
整体产能利用率从 78% 提升至 83%，单位固定成本摊薄。

然而，毛利率改善也面临两个制约因素。**一是工程渠道占比提升
带来的结构性拖累**——工程渠道毛利率约 28%，低于零售渠道的
38%，其占比从 35% 提升至 38% 拖累综合毛利率约 0.5 个百分点。
**二是物流成本上升**，国际海运费同比上涨 12%，部分抵消了原
材料降价的正面效果。

从可持续性角度看，毛利率有望在 2025 年维持在 33%-34% 区间。
支撑因素包括：智能产品线占比将继续提升（目标 20%），规模效应
将继续释放（新工厂产能爬坡）。制约因素在于：原材料价格已进入
底部区间，进一步下降空间有限；工程渠道的扩张可能持续拖累
毛利率。与行业龙头 4.8 个百分点的差距，短期内难以完全弥合，
主要因为品牌溢价（龙头高端产品占比 50% vs 公司 35%）和供应链
议价能力（龙头采购规模约为公司 3 倍）的差异。

【图 4-2：毛利率五年趋势及同业对比】
（图表解读：公司毛利率从 2020 年的 28.5% 稳步提升至 2024 年的
33.2%，五年累计提升 4.7 个百分点，改善趋势明确。但与行业龙头
（38.0%）仍有差距，与行业平均（29.5%）相比已处于中上水平。
差距主要来自产品结构和品牌溢价，预计随着智能产品占比提升，
差距将逐步收窄。）
```

## 八、调用方式

### 8.1 如何使用本技能

当需要撰写商业计划书的某个章节时，按以下流程调用：

```
Step 1：准备输入数据
  → 整理该章节需要的原始数据（结构化）
  → 明确章节类型和写作要求

Step 2：调用 business-writer
  → 输入：数据 + 章节类型 + 上下文信息
  → business-writer 根据本指南的框架生成内容

Step 3：质量自检
  → 对照"质量自检清单"逐项检查
  → 不合格的段落返回重写

Step 4：衔接校验
  → 检查与上一章节的衔接是否自然
  → 检查与下一章节的接口是否清晰
```

### 8.2 与其他技能的协作

```
business-writer 的上游输入：
  ├── finance-data-retrieval（财务数据获取）
  ├── finance（财务指标计算结果）
  ├── data（数据分析与可视化）
  └── baidu-search（行业洞察）

business-writer 的下游输出：
  ├── humanizer（最终润色）
  └── Word/PDF 文档生成（排版输出）
```

## 九、快速参考卡片

### 9.1 三层结构速查

```
What（25%）：数据事实 + 对比
Why（40%）：原因拆解 + 量化
How（35%）：深层含义 + 趋势
```

### 9.2 标题公式速查

```
序号 + 分析动作 + 分析对象 + 分析角度
示例：4.1 盈利能力改善：毛利率提升的可持续性分析
```

### 9.3 禁用词速查

```
❌ 综上所述、由此可见、总而言之
❌ 在当今...的大背景下、随着...的不断发展
❌ 公司是一家致力于...的领先企业

```

### 9.4 衔接词速查

```
递进：在...的同时、进一步看、更值得关注的是
转折：然而、但值得注意的是、与此形成对比的是
因果：因此、由此、基于以上分析
对比：与之相比、同业对比显示
展望：展望未来、基于以上趋势
```
