# Quick Reference - 快速参考卡片

## 三层结构速查

```
What（25%）：数据事实 + 对比
  → 当前数值 + 同比变化 + 行业对比

Why（40%）：原因拆解 + 量化
  → 分主次，先说主要原因
  → 每个原因量化贡献度

How（35%）：深层含义 + 趋势
  → 对公司战略/投资价值的影响
  → 未来趋势预判
```

## 标题公式速查

**公式**：序号 + 分析动作 + 分析对象 + 分析角度

**示例**：
- ✅ 4.1 盈利能力改善：毛利率提升的可持续性分析
- ✅ 5.2 国际化战略：从"出海"到"全球化"的关键跨越
- ✅ 6.1 客户集中度风险：前五大客户依赖度量化评估
- ❌ 4.1 财务分析（太空洞）
- ❌ 4.1 营收五年复合增长 18.5%（堆数据）

## 禁用词速查

### AI 味过渡词（禁用）
- ❌ 综上所述
- ❌ 由此可见
- ❌ 总而言之
- ❌ 不难看出
- ❌ 值得一提的是（过度使用）
- ❌ 在当今...的大背景下

### AI 味句式（禁用）
- ❌ "随着...的不断发展"
- ❌ "XX 具有广阔的发展前景"
- ❌ "XX 公司是一家致力于 XX 的领先企业"
- ❌ "公司将继续努力/不断进取"
- ❌ "公司管理层对未来发展充满信心"
- ❌ "XX 行业迎来了前所未有的发展机遇"

## 衔接词速查

| 功能 | 可用的衔接表达 |
|------|--------------|
| **递进** | 在...的同时、进一步看、更值得关注的是、无独有偶 |
| **转折** | 然而、但值得注意的是、与此形成对比的是、另一方面 |
| **因果** | 因此、由此、基于以上分析、这背后反映的是 |
| **总结** | 综合来看、总体而言、从以上分析可以得出 |
| **举例** | 具体来看、以...为例、从细分数据来看 |
| **对比** | 与之相比、同业对比显示、行业平均而言 |
| **展望** | 展望未来、基于以上趋势、如果这一态势延续 |

## 数据来源标注速查

- **年报数据**：（来源：公司 2024 年年报，第 X 页）
- **第三方数据**：（来源：艾瑞咨询《2024 年 XX 行业报告》）

- **同行对比**：（来源：各公司 2024 年年报，Wind 整理）

- **数据库**：（来源：Wind，家电行业数据库）

## 章节结构速查

### 执行摘要（300-500 字）
1. 公司定位（2-3 句话）
2. 财务亮点（3-4 句话）
3. 投资亮点（3-5 点）
4. 风险与建议（2-3 句话）

### 行业分析
- 市场规模段：规模 + 增速 + 驱动力 + 预测
- 竞争格局段：集中度 + 梯队划分 + 趋势

### 财务分析（重点）
1. 当前水平 + 对比（What）
2. 原因分析（Why）
3. 深层含义（How）
4. 趋势展望（可选）
5. 图表引用 + 解读（300 字+）

### 竞争分析
- 竞争格局概述
- 主要竞争对手分析（每个对手：地位 + 数据 + 优势 + 劣势 + 对比）
- SWOT 展开论述（禁止表格）
- 趋势判断

### 战略分析
- 现状分析
- 战略目标（SMART）
- 战略路径（具体举措 + 资源投入 + 时间节点）
- 风险与应对

### 风险分析
- 风险点识别（按重要性排序）
- 发生概率评估
- 影响程度量化
- 缓释措施
- 剩余风险 + 预案

## 常用分析框架速查

### 杜邦分析
```
ROE = 净利率 × 资产周转率 × 权益乘数
```
- 净利率：反映盈利能力
- 周转率：反映运营效率
- 权益乘数：反映财务杠杆

### 增长驱动力拆解
```
营收增量 = 因素 1 贡献 + 因素 2 贡献 + 因素 3 贡献
贡献度 = （因素增量 / 总增量）× 100%
```

### 毛利率变动归因
```
毛利率变动 = 产品结构优化贡献 + 原材料成本变动贡献 + 
            产能利用率提升贡献 + 其他因素
```

### 营运资金分析
```
现金转化周期 = 存货周转天数 + 应收账款周转天数 - 
              应付账款周转天数
```

## 写作技巧速查

### 消灭 AI 味
1. **避免"总 - 分 - 总"八股**：不要"首先...其次...最后..."
2. **避免过度使用"的"**：简化定语从句
3. **避免万能句式**：直接说事实，不用铺垫
4. **加入"不完美"表达**：承认风险和不确定性
5. **段落长短变化**：重点段落 8-10 行，过渡段 2-3 行

### 数据呈现技巧
1. **先结论后数据**：数据服务于观点
2. **对比呈现**：同比/环比/行业/竞品
3. **量化归因**：每个原因尽量量化贡献度
4. **来源标注**：增强可信度

### 逻辑推进技巧
1. **上一段结尾=下一段开头**：自然衔接
2. **从现象到本质**：What→Why→How
3. **从过去到未来**：历史→现状→展望
4. **从宏观到微观**：行业→公司→财务

## 典型场景速查

### 场景 1：营收增长分析
```
1. 增速对比：当前增速 vs 上年 vs 行业
2. 增量拆解：各业务/区域贡献度
3. 驱动归因：价格？销量？新品？并购？
4. 质量判断：有利润的增长 vs 牺牲利润换量
5. 趋势展望：未来增速预测 + 驱动因素
```

### 场景 2：毛利率分析
```
1. 水平对比：当前 vs 上年 vs 同业
2. 因素拆解：产品结构？原材料？产能利用率？
3. 量化贡献：每个因素贡献多少 pct
4. 可持续性：哪些因素可持续，哪些是一次性
5. 趋势判断：未来走势预测
```

### 场景 3：ROE 分析
```
1. 杜邦分解：净利率、周转率、杠杆率贡献
2. 深层归因：每个子因素的变化原因
3. 同业对比：与龙头的差距及原因
4. 质量判断：靠经营改善 vs 靠杠杆提升
5. 趋势展望：未来 ROE 预测
```

### 场景 4：竞争对手分析
```
1. 市场地位：营收、市占率、排名
2. 核心优势：1-2 点，量化
3. 主要劣势：1-2 点，量化
4. 与我方对比：强在哪/弱在哪
5. 竞争策略：对方可能的下一步动作
```

## 常用指标速查

### 盈利能力
- ROE（净资产收益率）
- ROA（总资产收益率）
- 毛利率、净利率
- EBITDA 利润率

### 营运能力
- 存货周转天数
- 应收账款周转天数
- 总资产周转率
- 现金转化周期

### 偿债能力
- 资产负债率
- 流动比率、速动比率
- 利息保障倍数
- 净负债率

### 成长能力
- 营收增长率
- 净利润增长率
- 总资产增长率
- CAGR（复合增长率）

### 现金流
- 经营现金流净额
- 自由现金流（FCF）
- 销售现金比率
- 现金回笼率
